# Project AXIS

A modular hybrid locomotion and cinematic robotics system.

## The Problem
Current robots are single-mode — wheeled robots fail on rough terrain, bipedal robots are hard to balance, aerial drones can't operate at ground level. No single system handles all three environments.

## The Solution
AXIS combines roller-assisted bipedal legs with a detachable autonomous drone head — switching modes based on real-time terrain sensing.

## Current Status
Design Study v2.0 — Conceptual Architecture complete.

## Next Steps
